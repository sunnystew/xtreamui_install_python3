# XtreamUI Installer (Python 3)
This is an installer for XtreamUI, written in Python 3 instead of the already available scripts which all use the old and deprecated Python 2. The reason I made this script is that Python 2 is not really provided anymore on Linux distros and/or provided but with limited support. By migrating the installer to Python 3, I've effectively made it installable on modern Linux.

# How to install
To install XtreamUI, simply run:
```
git clone https://github.com/sunnystew/xtreamui_install_python3
cd xtreamui_install_python3
python3 install_new.py
```
